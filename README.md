# SafeEye - AI Powered Crowd Management System

## Overview

SafeEye is an AI-based crowd monitoring system that detects overcrowding in real time using computer vision. When the crowd exceeds a predefined limit, the system generates an alert and recommends deploying nearby security personnel to the affected zone. The prototype uses a live mobile camera feed and is designed for future CCTV integration.

## Features

- Real-time crowd detection
- Crowd counting using AI
- Multi-zone monitoring
- Automatic alert generation
- Security allocation recommendation
- Live dashboard

## How It Works

1. Capture live video from the camera.
2. Detect people using YOLOv8.
3. Count the number of people.
4. Compare the count with the threshold.
5. Generate an alert if the limit is exceeded.
6. Display results on the dashboard.
